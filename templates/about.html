The Messi Vs Ronaldo StatBot is an interactive web application designed to provide football fans with comprehensive statistics and insights about two of the greatest football players of all time: Lionel Messi and Cristiano Ronaldo.

Using a conversational interface, users can ask questions about either player's career achievements, records, performance statistics, and receive instant, accurate responses backed by a comprehensive database of football statistics.

Features

  • AI-Powered Conversations

    Ask questions in natural language about either player and get detailed, accurate responses.

  • Comprehensive Statistics

    Access detailed statistics spanning both players' entire careers, regularly updated with new data.

  • Dark Mode Support

    Toggle between light and dark themes for comfortable viewing in any environment.

  • Responsive Design

    Enjoy a seamless experience across desktop, tablet, and mobile devices.

Technology Stack

This project was built using modern web technologies to ensure high performance, accessibility, and a great user experience:

HTML5 CSS3 JavaScript Python Web-Scrapping via beautifulsoup Natural Language Processing
